\item \underline{Task}:
	\begin{itemize}
	\item Design a biaxial tensile specimen suitable for the validation of anisotropic damage models for sheet moulding compound (SMC).
		Such a specimen has to result in damage and failure in the biaxially loaded, central specimen area.
	\end{itemize}
\visible<3->
{
\item \underline{Approach}:
	\begin{itemize}
	\item Multiobjective shape optimisation:
		\begin{itemize}
		\item Create a parametric model as a function of a parameter vector $\vp$.
		\item Formulate objective functions, measuring the appropriateness of a given specimen.
		\item Find a parameter vector $\vp$, minimising the objective functions.
		\end{itemize}
	\end{itemize}
